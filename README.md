# shiny-app-project

## Overview
This project explore the health disparity data from CHIS. 

## Requirements
This project requires the following packages:
`tidyverse`
`corrplot`
`caret`
`cowplot`
`shiny`
`shinyWidgets`
`shinythemes`
`DT`
`plotly`
`imager`.
